## ⚡ HIZLI BAŞLANGIÇ (5 Dakika)

### 1. Website'i Aç
```bash
# Klasöre git
cd "C:\Users\Admin\Desktop\pergo clean"

# Tarayıcıda aç
start index.html
```

### 2. Görselleri Ekle
1. `images/` klasörü oluştur
2. [GÖRSEL_REHBERİ.md](./GÖRSEL_REHBERİ.md)'yi oku
3. Hero ve ürün görsellerini ekle

### 3. Logo Yerleştir
1. [LOGO_HAZIRLAMA.md](./LOGO_HAZIRLAMA.md)'yi takip et
2. SVG logo oluştur
3. index.html'de logo'yu değiştir

### 4. n8n Entegre Et
1. [N8N_WEBHOOK_KURULUM.md](./N8N_WEBHOOK_KURULUM.md)'u oku
2. n8n hesabı oluştur
3. Webhook'ları kurulumu yap
4. script.js'i güncelle
