🔗 N8N WEBHOOK READY:

Form submission kodunu n8n webhook'a bağlamak için:

```javascript
// script.js içinde form submission kısmını değiştir:

fetch('https://n8n.tozyapi.com/webhook/contact-form', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify(formData)
})
.then(response => response.json())
.then(data => {
  showNotification('Mesajınız alındı!', 'success');
})
.catch(error => {
  showNotification('Bir hata oluştu.', 'error');
});
```
